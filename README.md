# Language Translation Tool

This project is a Streamlit-based language translation tool built using
LangChain and OpenAI's GPT model.

## Features
- User-friendly UI
- Source & target language selection
- Real-time translation using OpenAI API
- Supports multiple languages

## Technologies Used
- Python 3.10
- Streamlit
- LangChain
- OpenAI API

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Add your OpenAI API key in .env file
3. Run:
   streamlit run translation_tool.py
